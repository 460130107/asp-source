# Écoute

##
Les criquets stridulent.

##
Les souris couinent.

##
Les serpents sifflent.

##
Les chiens aboient.

##
Les chèvres bêlent.

##
Les vaches meuglent.

##
Les lions rugissent…

##
…et les escargots ne
font aucun bruit.

##
Les bébés babillent…

##
…et les enfants parlent.
