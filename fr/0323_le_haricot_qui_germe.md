# Le haricot qui germe

##
La racine pousse la
première.

##
Elle pousse vers le bas
dans le sol.

##
Puis la tige pousse. Elle
pousse vers le haut,
vers le soleil.

##
Les premières feuilles
sortent.

##
La plante à haricots
devient haute.

##
Puis les fleurs sortent.
Elles se transforment
en haricots.

##
Il y a beaucoup de
graines dans les
haricots.
