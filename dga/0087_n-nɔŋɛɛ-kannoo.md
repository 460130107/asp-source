# N Nɔŋɛɛ Kannoo

##
N nɔŋɛɛ kannoo.

##
N kanne ko aŋ?

##
N yɔɔpɔge gbiri la.

##
N kanne ko aŋ?

##
N ma ane N makoma meŋ tona la tona.

##
N kanne ko aŋ?

##
N saa ane N saakoma meŋ tona la tona.

##
N kanne ko aŋ? N kanne ko N meŋa bee?
