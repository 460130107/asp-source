# I can do this

##
I can do this.
I can hop.

##
I can do this.
I can jump.

##
I can do this.
I can dance.

##
I can do this.
I can build.

##
I can do this.
I can wink.

##
I can do this.
I can sing.

##
I can do this.
I can play.

##
I can do this.
I can read.
